# terminal-todo-app

terminal-todo-app is a simple todo app for the terminal. It is written in javascript and node.js.

## Installation

```bash
git clone git@github.com:brandonpaldom/terminal-todo-app.git
```

## Usage

```bash
cd terminal-todo-app

npm install

node app.js
```
